# Employee Management System

## Project Overview

Employee Management System is a Full Stack Web Application developed using ASP.NET Core Web API and Angular.

The system provides role-based authentication and allows administrators to manage departments and employees efficiently.

## Features

### Authentication

* JWT Authentication
* Role Based Authorization
* Admin Login
* User Login

### Department Management

* Add Department
* View Department List

### Employee Management

* Add Employee
* Update Employee
* Soft Delete Employee
* View Employee Details

### Dashboard

* Total Employees
* Active Employees
* Inactive Employees
* Total Departments
* Department Wise Employee Chart

### Reports

* Department Wise Employee Report

### User Profile

* Employee Profile View

## Technologies Used

### Backend

* ASP.NET Core Web API
* C#
* ADO.NET
* SQL Server
* Stored Procedures
* JWT Authentication

### Frontend

* Angular
* TypeScript
* Bootstrap 5
* SweetAlert2

### Database

* SQL Server

## Project Structure

EmployeeManagement -> ASP.NET Core Web API

EmployeeManagementUI -> Angular Frontend
